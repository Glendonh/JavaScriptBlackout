Blackout Rules

Select a number below. The program will list all of the numbers from 1 to the number of your choice, unless they are divisible by 3 and/or 5. If the number is divisible by 3 it will be replaced by "Black". If the number 5 it will be replaced by "Out". If the number is divisible by 3 and 5 it will be replaced by "Blackout"
